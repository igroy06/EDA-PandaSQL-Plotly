# EDA with Pandasql and Plotly on Data Salaries
The 2024 dataset on data developer salaries and employment attributes offers valuable insights into the evolving landscape of data developers. It includes key variables such as salary, job title, experience level, employment type, employee residence, remote work ratio, company location, and company size. Researchers, analysts, and organizations can leverage this dataset to better understand compensation trends, the distribution of data developer roles across different regions, and the impact of remote work and company size on employment in this field. The notebook has a detailed exploratory data visualization of salary trends, employment patterns, and geographic variations in data developer roles. 

The objective is querying using pandasql and visualize the same with plotly on salaries
